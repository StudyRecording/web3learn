## 核心要点

### 以太坊是什么

以太坊是一个**去中心化的全球计算机**，允许任何人在其上运行程序（智能合约）。

```
┌─────────────────────────────────────────────────────────────┐
│                     以太坊架构                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  用户层                                                      │
│  ├── DApp 前端                                              │
│  └── 钱包 (MetaMask)                                        │
│                                                             │
│  接口层                                                      │
│  └── JSON-RPC API                                          │
│                                                             │
│  节点层                                                      │
│  ├── 执行客户端 (Geth, Nethermind, Besu)                   │
│  └── 共识客户端 (Prysm, Lighthouse, Teku)                   │
│                                                             │
│  核心层                                                      │
│  ├── EVM (以太坊虚拟机)                                     │
│  ├── 状态数据库                                             │
│  └── 共识机制 (PoS)                                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 以太坊 vs 比特币

| 特性 | 比特币 | 以太坊 |
|------|--------|--------|
| 主要用途 | 价值存储/支付 | 智能合约平台 |
| 编程能力 | 有限脚本语言 | 图灵完备 (Solidity) |
| 区块时间 | ~10 分钟 | ~12 秒 |
| 共识机制 | PoW | PoS (合并后) |
| 代币 | BTC | ETH |
| 应用 | 转账 | DeFi, NFT, DAO 等 |

### ETH 单位

```
ETH 单位换算

1 ETH = 10^18 Wei (最小单位)

常用单位:
├── Wei:     1 (最小单位)
├── Gwei:    10^9 Wei (Gas 价格单位)
├── Ether:   10^18 Wei
└── ETH:     1 Ether

示例:
1 ETH      = 1,000,000,000,000,000,000 Wei
1 Gwei     = 1,000,000,000 Wei
0.001 ETH  = 1,000,000,000,000,000 Wei
```
