## 中级项目

### 1. 借贷协议

类似 Aave/Compound 的借贷协议：

```
核心功能:
├── 存款赚取利息
├── 抵押借贷
├── 利率模型
├── 清算机制
└── 闪电贷

技术要点:
├── 利率曲线设计
├── 抵押率计算
├── 清算阈值
└── 健康因子

学习资源:
├── Compound 白皮书
├── Aave 白皮书
└── GitHub 开源代码
```

### 2. 跨链桥

实现资产在不同链之间转移：

```
核心功能:
├── 锁定/铸造模式
├── 验证者网络
├── 消息传递
└── 安全验证

技术要点:
├── 多签验证
├── 中继网络
├── 事件监听
└── 重放攻击防护

学习资源:
├── LayerZero 协议
├── Wormhole
└── Multichain
```

### 3. 收益聚合器

自动寻找最优收益策略：

```
核心功能:
├── 多协议集成
├── 自动复投
├── 策略切换
└── 收益追踪

技术要点:
├── 策略合约设计
├── 收益计算
├── 费用抽取
└── 安全审计

学习资源:
├── Yearn Finance
├── Beefy Finance
└── 相关文档
```

## 高级项目

### 1. 永续合约交易所

类似 dYdX/GMX 的去中心化永续合约：

```
核心功能:
├── 杠杆交易
├── 做市机制
├── 预言机集成
├── 清算系统
└── 资金费率

技术要点:
├── 订单簿 vs AMM
├── 预言机延迟攻击防护
├── 清算机器人
└── 跨保证金

学习资源:
├── GMX 文档
├── dYdX 文档
└── Perp Protocol
```

### 2. 流动性质押

类似 Lido 的流动性质押协议：

```
核心功能:
├── 质押 ETH
├── 发行 stETH
├── 验证者管理
└── 奖励分配

技术要点:
├── 信标链交互
├── 提款凭证
├── 验证者集
└── slashing 风险

学习资源:
├── Lido 文档
├── Rocket Pool
└── 以太坊质押文档
```

### 3. Layer 2 开发

使用 OP Stack 构建自定义 L2：

```
核心功能:
├── 自定义 Gas 代币
├── 自定义区块时间
├── 自定义验证者
└── 数据可用性层

技术要点:
├── Optimistic Rollup 原理
├── 欺诈证明
├── Sequencer 管理
└── L1<>L2 消息传递

学习资源:
├── Optimism 文档
├── Arbitrum 文档
└── Base 文档
```
